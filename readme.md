# 🌱 WebTEA

Plataforma web de orientação e acolhimento para pessoas recém-diagnosticadas com Transtorno do Espectro Autista (TEA) e seus familiares, com foco no estado do Amapá.

> Desenvolvido como projeto acadêmico.

---

## Sobre o projeto

Receber um diagnóstico de TEA — para si mesmo ou para alguém próximo — costuma vir acompanhado de muitas dúvidas e pouca orientação clara. O WebTEA foi criado para ser o primeiro passo: um espaço digital acolhedor que organiza informações confiáveis, indica os próximos passos práticos e conecta pessoas por meio de experiências compartilhadas.

O design prioriza acessibilidade cognitiva: tipografia legível, cores suaves, sem sobrecarga visual, e conteúdo apresentado de forma progressiva — respeitando o ritmo de cada pessoa.

---

## Funcionalidades

- **Trilha A — Para você:** voltada para quem foi diagnosticado, com validação emocional, primeiros passos práticos, direitos garantidos por lei e passo a passo para emissão da Carteira TEA no Amapá
- **Trilha B — Para quem apoia:** voltada para familiares e pessoas próximas, com orientações sobre como processar a notícia, apoiar sem capacitismo e conduzir o tratamento do dependente
- **Mural de Conselhos:** espaço para pessoas diagnosticadas compartilharem o que gostariam de ter ouvido no início da jornada
- **Vivências:** espaço para familiares e cuidadores compartilharem suas experiências
- **Progresso automático:** seções lidas são salvas automaticamente, sem necessidade de ação manual
- **Conteúdo localizado:** direitos, serviços e endereços específicos do estado do Amapá

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/webtea.git

# Acesse a pasta
cd webtea

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Acesse em `http://localhost:5173` (ou a porta indicada pelo Vite).
